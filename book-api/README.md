# React side Project

## Issue
- 네이버 openAPI로 도서 검색 후 나의 서재에 책 추가 및 삭제
- 나의 서재에 이미 있는 책인 경우 추가 불가능
- 라디오 버튼으로 책 선택 후 + 버튼을 누르면 나의 서재에 책 추가
- 나의 서재에서 x 버튼으로 책 삭제

## Getting Started
1. Install all dependencies
2. Run

```bash
yarn start
```
